# SQLZoo 练习记录

本目录为使用 [SQLZoo](https://sqlzoo.net/) 平台进行的 SQL 语法与查询实战练习。  
所有题目内容均为 **平台上的学习题意个人转述表达**，**不包含平台原始题干文本**。  
仅保留练习编号与个人编写的 SQL 代码，便于复盘与技能巩固。

---

## 📁 文件命名规则

- 按照 SQLZoo 平台的模块编号命名，如：
  - `0_select_basics.sql`：代表  模块 0 的练习
  - `1_select_name.sql`：代表  模块 1 的练习
